# **App Name**: Cupping Compass

## Core Features:

- SCA Data Upload: Allow users to upload SCA cupping evaluation files (e.g., CSV, Excel).
- Data Transformation: Automatically parse and transform uploaded SCA data into a consistent format for analysis and display.
- Interactive Visualizations: Display cupping results using interactive charts and tables for easy understanding and analysis.
- Report Generation Tool: Allow users to generate comprehensive cupping reports from transformed data, ready for sharing. Use the uploaded data to generate useful text summaries of the results to include in the report.
- Session Organization: Enable users to organize cupping evaluations into sessions for comparison and tracking.
- Shareable Reports: Provide a function to create shareable links for cupping reports.

## Style Guidelines:

- Primary color: Warm tan (#D2B48C) to evoke a sense of earthiness and comfort.
- Background color: Light beige (#F5F5DC), a desaturated variant of the primary, providing a neutral and warm backdrop.
- Accent color: Muted brown (#A67B5B) for interactive elements and highlights, offering a contrasting, yet harmonious touch.
- Font: 'PT Sans', a humanist sans-serif font suitable for both headlines and body text, balancing modernity with approachability.
- Use clean, simple icons related to coffee and cupping, maintaining a consistent style throughout the app.
- Employ a clean and intuitive layout with a focus on usability and accessibility of data visualizations.
- Use subtle transitions and animations to enhance user experience, such as chart updates and data loading indicators.
